SERVICIOS WEB
==================================================================================================

Principales:
- AWS (Amazon - Amazon Web Services)

    Servicios sobresalientes
    
    1. Amazon EC2
        Al iniciar con una sesion se puede acceder a un servidor para usarlo como uno desee
    
    2. Amazon RDS (Amazon Relational Database Service)
        Servicio de BD relacional evitando asi la renta del quipo de servidor instalacion de BD configuracion y manto,
        servicio de pago que brinda las capacidades de una BD relacional (sql como lenguaje de consulta)
    
    3. Amazon IAM (Aws Identity and Acces Managment)
        Gestor de indentidad y acceso, configura un usuaio y asigna permisos a otros usuarios par acceder a tu servicio
        generalmente usado en equipos grandes de trabajo
    
    4. Amazon S3 (Simple Storage Service)
        Servicio de almacenamiento simple, servicio de almacenamiento en la nube
