ESTRUCTURA DE DATOS
==================================================================================================
Estructura para resolver problemas de programacion

1. Tabla Hash
Objeto en js o diccionario en python, permite acceder a elementos rapidamente, solo se requiere de una llave 
para acceder a ese dato  en el codigo 

2. Lista
Guarda datos ordenadamente, para acceder a los datos se peude hacer uso de bucles 

3. Pila (stack - lifo, last in first out)
Solo se puede ingresar y sacar datos en la parte superior

4. Cola (fifo, first in first out)
Se puede usar en servidores que operan a base de eventos 

5. Lista enlazada
Los elementos no se guardan de forma continua en la memoria sino como nodos en diferentes
partes de la memoria, como elementos enlazados, ej. funcion tab en una pagina web

6. Arbol
A base de nodos, un nodo raiz y esta conectada a otros nodos, ej. el DOM o HTML (html -> head y body)
importante saber recorrer un arbol, algnos metodos son: pre-orden, orden o post-orden, metodo cola para recorrer 
los elementos de un arbol 

7. Arbol binario
Igual que un arbol pero solo con 2 hijos, para buscar elementos en esta estructura se recomienda usar un
'arbol binario de busqueda'    

8. Arbol binario de busqueda
Arbol binario donde los hijos (elementos) de la izquierda de cada nodo tiene que ser menor al nodo padre 
y el hijo (elemento) de la derecha debe ser mayor al nodo, muy eficiente para ser busqueda ej. indices de Bases de datos

9. Grafos 
Permite representar una forma infinita de elementos o nodos que pueden estar conectados entre si de diferetes maneras.
ej. ciudades conectadas por carreteras o conexiones en redes sociales entre amigos

10. Heap
Parecido a un arbol pero con nivel de prioridades, al ingresar un nuevo dato se reordena segun su prioridad
